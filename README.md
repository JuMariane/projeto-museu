# ✨ Museu da Julia ✨

Bem-vindos ao meu cantinho especial! Este é o meu **Museu de GIFs**, um projeto desenvolvido com muito carinho para colocar em prática meus conhecimentos no curso técnico de **Desenvolvimento de Sistemas**. 👩‍💻💖

---

## 🎀 Sobre o Projeto

O **Museu da Julia** é um espaço interativo onde reuni GIFs "peculiares" e fofinhos de gatinhos, cachorros e coelhos. O objetivo foi criar uma experiência leve, colorida e divertida para quem visita.

Este projeto faz parte da minha jornada no **terceirão** da Escola Estadual Ruy Rodriguez, em Campinas! 🏫💫

---

## 🛠️ O que tem de especial? (Funcionalidades)

Durante o desenvolvimento, foquei em três atividades principais para deixar o site incrível:

* **🖼️ Galeria Dinâmica:** Na página de gatos, temos um botão mágico que troca a obra de arte (GIF) sem precisar atualizar a página!
* **🔙 Navegação Intuitiva:** Todas as salas do museu possuem um botão de "Voltar ao Início" personalizado para você nunca se perder.
* **🎨 Design Harmonioso:** Usei um CSS todo trabalhado em tons pastéis e gradientes para deixar tudo com a minha cara.
